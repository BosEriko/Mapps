## Expo Setup

Install Expo globally

  npm install -g expo-cli

Before running your app on iOS, make sure you have CocoaPods installed and initialize the project:

  cd MappsExpo/ios
  pod install

Then you can run the project:

  cd MappsExpo
  yarn android
  yarn ios
